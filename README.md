### 🏎️ How to Run all Test Scenarios
    > Manual Testing using Postman:
      a. Add new request
      b. Choose method
      c. Input url
      d. Input body (if needed)
      e. Input assertion
      f. Click send

    > Automation Testing:
      a. Click on text editor's terminal
      b. input => mvn clean verify
         or:
      a. You can click run from CucumberTestSuite
